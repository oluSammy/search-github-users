# Todo Application

GitHub User Search

## Description

an app that re-implements a portion of GitHub's Search feature, the user search, using their public API

## Installation

In the root directory

```bash
  yarn
  yarn start
```

## Author

- [@olusammy](https://www.github.com/olusammy)

## Tech Stack

**Client:** React, Typescript

**API:** Github API
